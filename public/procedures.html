Procedimiento

Conjunto de sentencias con un nombre, que podemos ejecutar usando dicho nombre.

Definición vs Llamada

Sintaxis de definición de un procedimiento

Sintaxis de llamada a un procedimiento

Ejemplo

#include "ticlib.h"

int main() {
  for (int i {0}; i < 5; ++i) {
    for (int j {0}; j < 5; ++j) {
      cout << "*";
    }
    cout << "\n";
  }
}
#include "ticlib.h"

void dibuja_fila()
{
  for (int j {0}; j < 5; ++j) {
    cout << "*";
  }
  cout << "\n";
}

int main() {
  for (int i {0}; i < 5; ++i) {
    dibuja_fila();
  }
}

Variables locales

Las variables creadas en un procedimiento solo son visibles dentro del procedimiento

Variables locales

#include "ticlib.h"

void get_user_name()
{
  cout << "Username: ";
  string username;
  cin >> username;
  cout << "Welcome, " << username << "\n";
}

int main() {
  get_user_name();
  if (username == "anonymous") { // ERROR: username no es visible
    cout << "Anonymous users are not allowed.\n";
  }
}
